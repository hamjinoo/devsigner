How it Works

Three steps. Zero design skills required.

devsigner is an MCP server. Connect it to your AI coding tool and ask naturally.

1

Connect

Add devsigner to your MCP client config. One line, no API keys.

{ "mcpServers": { "devsigner": { "command": "npx", "args": ["-y", "devsigner"] } } }
2

Ask

Talk to your AI naturally. devsigner tools activate automatically.

// Just ask in plain English
"Review my code for design issues"
"Generate a pricing page"
"Fix all the spacing problems"
"Run an accessibility audit"
3

Ship

Get production-ready results. Scored, validated, copy-pasteable.

// Before Score: 23/100 // After Score: 89/100
+ padding: 13px -> 16px + contrast 1.66 -> 4.7:1 + font scale normalized

Design Identity

Six personality archetypes.

Each archetype defines typography, colors, spacing, motion, and tone. Pick one, or describe your product and let devsigner choose.

Bold Minimal

Linear, Vercel

Extreme restraint with one bold punch. The power of emptiness -- what you don't show is as important as what you show.

Generous whitespace Muted color Subtle corners Confident type No shadows

Warm Professional

Stripe, Mercury

Trustworthy but approachable. Feels expensive without being cold. The gradient is the handshake.

Balanced spacing Medium color Rounded corners Subtle shadows Gradient accents

Energetic Pop

Duolingo, Discord

Bold colors, playful interactions. Makes you smile. Design should feel like a reward, not a chore.

Bold color Pill corners Dramatic shadows Playful type Expressive motion

Elegant Editorial

Notion, Medium

Typography IS the design. Serif and sans-serif pairing creates instant sophistication. The words are the interface.

Generous whitespace Sharp corners Elegant type No motion Muted color

Data Dense

GitHub, Grafana

Maximum information, minimum noise. Respect for the user's expertise. Don't waste their time with fluff.

Dense spacing Functional color Compact layout Monospace data Neutral type

Soft Wellness

Calm, Headspace

Calming, accessible, human. Design as a deep breath. Everything says "take your time, you're safe here."

Generous whitespace Pastel colors Pill corners Friendly type Soft shadows
